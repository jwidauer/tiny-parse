Detailed Description

template<class T, class S>
class tiny_parse::Or< T, S >

A parser that matches one parser or the other.

First tries the first parser and if it fails, tries the second parser. If both fail, the result is a unsuccessful parse.

Template Parameters
TThe first parser that will be tried.
SThe second parser that will be tried.
